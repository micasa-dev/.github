Your house is quietly plotting to break while you sleep &mdash;
and you're dreaming about redoing the kitchen.

**[micasa](https://github.com/micasa-dev/micasa)** tracks maintenance, projects, incidents, appliances,
vendors, quotes, and documents from your terminal.
Single SQLite file. No cloud. No account. No subscriptions.
Vim-style modal keys. Keyboard driven. Runs everywhere.

```
brew install micasa && micasa demo
```
