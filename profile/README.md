## micasa

Your house is quietly plotting to break while you sleep -- and you're dreaming about redoing the kitchen.

**[micasa](https://github.com/micasa-dev/micasa)** tracks maintenance, projects, incidents, appliances, vendors, quotes, and documents from your terminal. Single SQLite file. No cloud. No account. No subscriptions.

### Get started

```sh
brew install micasa        # Homebrew (macOS/Linux)
micasa demo                # poke around with sample data
```

Or grab a binary from the [latest release](https://github.com/micasa-dev/micasa/releases/latest). Full install options in the [docs](https://micasa.dev/docs/getting-started/installation/).
